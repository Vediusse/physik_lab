у нас есть двухмерное тело прямоугольной формы

1) Пользователь задает числа A и B - это форма прямоугольника
2) Пользователь задает число n
3) Пользователь задает два числа, которые являются делителями числа n и при умножении должно получиться n

Тело слева имеет температуру 0 градусов по цельсию, справа температуру A
Температура снизу и сверху задаётся формулой (в питоне это lambda выражение) снизу пускай будет x, а сверху (x^2)/A

Задача программы посчитать температуру в n-точках - выбирать их стоит так: мы указали в пункте 3 количество точек - наибольшее мы равномерно расставим по горизонтали с шагом (d = n/количество точек) так же сделаем и по вертикали. А также поставим на краях тела такие же точки (в этих точках мы уже знаем температуру)
Как считать температуру?
У нас для каждой точки внутри тела есть уровнений

dT/dx + dT/dy = 0

где dT/dx для какой то точки по координатам i,j
dT/dx = (T_i+1,j + T_i-1,j - 2* T_i,j)/(d^2)

то есть для точки i,j мы берем сумму температур правой и левой точки, потом вычитаем удвоенную температуру в самой точке и делим на квадрат шага между правой точкой и самой точкой

По аналогии делаем и для dT/dy, только там точки по вертикале то есть мы берем точки ниже и выше точки

В результате мы сделаем n уравнений ввида dT/dx + dT/dy = 0 с n неизвестными
